# Gin + Svelte

* Back-End: Gin
* Front-End: Svelte
* UI: Tailwind CSS

---

## Directory: /front-end

Run Svelte Dev Server
```
$ npm run dev
```

Run Watch Tailwind CSS
```
$ npm run watch:style
```

---

## Directory: /

Run Gin Server
```
$ go run cmd/main.go
```

Run All Dev Server (includes front-end)

```
$ npm run dev
```

The server will run at http://localhost:8000
